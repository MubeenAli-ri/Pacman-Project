1. Pacman Console Game in Java

This is a simple terminal-based Pacman game developed in Java using Object-Oriented Programming (OOP). The game runs entirely in the console. The player controls Pacman using keyboard inputs to collect food and avoid a ghost.

---

2. Project Overview

The goal of the game is to collect all food items on the board while avoiding the ghost. The game ends when either all food is collected or Pacman collides with the ghost.

Controls:
- > Move right  
- < Move left  
- ^ Move up  
- v Move down  

---

3. Project Structure

The project is divided into multiple classes that work together:
- Pacman (Main Class)
- GameBoard
- Player
- Ghost
- Food

Each class interacts with others to manage movement, game state, and rendering of the board.

---

4. How to Run

Compile: javac Pacman.java
Run: java Pacman

---

5. Conclusion

This project demonstrates basic object-oriented programming in Java through a simple console-based game where multiple classes interact to create a working system.
